# Circles mini app submission checklist

Per https://docs.aboutcircles.com/miniapps (embedded mini apps need SDK
integration + store listing; standalone needs QR/tx-URL flows).

## Materials in this repo
- [x] One-pager (`ONE-PAGER.md`)
- [x] Architecture & security docs (`docs/`)
- [x] FAQ for reviewers (`FAQ.md`)
- [x] Milestones + grant ask (`MILESTONES.md`)
- [x] Working standalone demo: catalog + CRC quotes against live APIs

## To prepare before submitting
- [ ] Deployed demo URL (standalone mode) — e.g. giftcards.hfsp.xyz
- [ ] App icon + 3–5 screenshots (Circles store listing assets)
- [ ] Short demo video (browse → quote → pay via QR → code delivered, testnet)
- [ ] Circles SDK embedded-mode integration branch
- [ ] Contact + support channel listed (info@hfsp.xyz, hfsp.xyz)
- [ ] Terms of service + privacy policy page (codes retention, refunds)

## Who to contact
- Circles team via docs.aboutcircles.com / About Circles community channels
- Gnosis ecosystem/grants (GnosisDAO forum post referencing this repo)
- Cryptorefills partner contact via their agentic-commerce repo / website
